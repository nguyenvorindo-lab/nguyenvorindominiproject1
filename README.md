# Mini-project 2: HTML Head Checklist

Dự án hoàn thiện trang web hướng dẫn cấu hình phần Head, định dạng mã nguồn và tổ chức dự án bài bản trên GitHub.

## 📁 Cấu trúc thư mục dự án
- `index.html`: File cấu trúc chính của trang web, phân vùng rõ ràng.
- `css/style.css`: File chứa cấu hình phong cách (đang để trống theo tiến độ bài học).
- `js/script.js`: File chứa kịch bản JavaScript xử lý logic (đang để trống).
- `assets/ui-design.png`: Ảnh chụp bản vẽ thiết kế UI phân vùng giao diện trước khi code.

## 🛠️ Phân vùng HTML theo thiết kế UI
Trang web được cấu trúc chặt chẽ dựa trên bản vẽ phối cảnh thiết kế nằm trong thư mục `assets/`, bao gồm 3 phân vùng chính:
1. `<header>`: Khu vực hiển thị tiêu đề cẩm nang.
2. `<main>`: Khu vực chứa nội dung các bài học checklist và hướng dẫn debug tài nguyên.
3. `<footer>`: Khu vực thông tin bản quyền chân trang.
