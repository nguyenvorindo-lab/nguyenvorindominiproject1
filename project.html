Hướng dẫn cấu hình HTML Head và Tổ chức bài bản

1. Các thành phần bắt buộc trong Head

  • Charset: Định dạng mã hóa UTF-8 để hiển thị tiếng Việt chuẩn.
  • Viewport: Giúp trang web hiển thị responsive trên di động.
  • Title: Tiêu đề hiển thị trên tab trình duyệt.
  • Link CSS: Liên kết tệp trang trí bên ngoài.

2. Quy tắc định dạng và kiểm tra

Mã nguồn cần được viết nhiều dòng, thụt lề hợp lý bằng phím Tab để phân biệt thẻ cha-con. Sử dụng DevTools (tab Console) để kiểm tra nếu đường dẫn CSS hoặc JS bị lỗi 404.
